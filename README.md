🎤 Voice Chatbot System
📌 Project Description

This is an interactive web-based chatbot application that supports both voice and text input for seamless user interaction. The chatbot initiates the conversation with a friendly greeting and asks for the user's name to personalize responses. It leverages browser-based Speech Recognition for voice input and Speech Synthesis for voice output, creating a natural conversational experience. The interface features a clean and user-friendly chat window with smooth scrolling and styled message bubbles for clear distinction between user and bot messages.


🚀 Features
Speech-to-Text (Voice input)

AI response generation

Text-to-Speech output

Real-time conversation

Simple web interface (HTML, JS)

Personalized greetings and context-aware replies


🛠️ Technologies Used

HTML

JavaScript

Web Speech API

▶️ How to Run

Clone or download the repository

Open index.html in a modern web browser (preferably Chrome or Edge)

Click Speak to start voice input or type a message and click Send

Interact with the chatbot via voice or text
