his is an interactive web-based chatbot application that supports both voice and text input for seamless user interaction. The chatbot initiates the conversation with a friendly greeting and asks for the user's name to personalize responses. It leverages browser-based Speech Recognition for voice input and Speech Synthesis for voice output, creating a natural conversational experience. The interface features a clean and user-friendly chat window with smooth scrolling and styled message bubbles for clear distinction between user and bot messages.

Key Features:

Supports both text and voice input for flexibility.

Personalized interactions by remembering the user's name.

Real-time speech-to-text conversion using Web Speech API.

Text-to-speech responses for a more engaging experience.

Simple, responsive, and visually appealing chat interface.

Basic AI logic for conversational replies with easy extensibility
